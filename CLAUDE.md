## MCP Integration
This project uses the **Orgo MCP server** for cloud computer automation.

Key Orgo tools available:
- `orgo_create_computer` - Spin up cloud VMs
- `orgo_screenshot` / `orgo_click` / `orgo_type` - GUI automation
- `orgo_bash` / `orgo_exec` - Run commands/Python on remote machines
- `orgo_prompt` - Launch autonomous AI agents on computers
